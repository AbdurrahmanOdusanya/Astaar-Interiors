##  About the Project

**Astaar Interior Decoration** delivers custom, high-end interior styling solutions, specializing in luxury curtains and home transformations. This web platform is built to represent the brand's sophisticated aesthetic online. It provides a visual gallery of design projects, premium fabrics, and elegant decor styling to inspire clients and streamline their design inquiries.

> _Your space, transformed with elegance and precision._

---

##  Features

-  **Luxury Portfolio Gallery:** A visually stunning showcase of completed interior and curtain installations
-  **Service Overview:** Complete details of curtain custom tailoring, window consultations, and decor staging
-  **Responsive Architecture:** Fully optimized for seamless browsing on desktops, tablets, and smartphones
-  **Elegant UI/UX:** Clean layouts, high-quality typography, and design elements matching a luxury brand identity
-  **Direct Inquiry Channel:** Integrated navigation to guide users to book design consultations effortlessly

---

##  Technologies Used

- **TypeScript:** Ensures reliable, structured, and type-safe frontend code
- **HTML5 & CSS3:** For responsive UI grids, fluid transitions, and layout structure
- **Netlify:** Rapid deployment, edge network hosting, and automated continuous delivery

##  Getting Started

### Prerequisites

Ensure you have **Node.js** installed on your computer.
👉 [Download Node.js](https://nodejs.org/)

### Installation

1. **Clone this repository**

   ```bash
   git clone https://github.com/your-username/astaar-interior-decor.git
   ```

2. **Move into the project root directory**

   ```bash
   cd astaar-interior-decor
   ```

3. **Install the required development packages**

   ```bash
   npm install
   ```

4. **Fire up the local development environment**

   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:3000`
